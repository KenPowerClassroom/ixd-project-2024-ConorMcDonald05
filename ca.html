Competitive Audit

Question Kahoot Code Combat Typing.com
Engagement: How engaging is the app? Is it visually appealing? Does it have sound and animation to capture the user's attention? Does it have a good balance of challenge and reward? Kahoot makes the most of gamification by integrating a variety of unique gaming features into the application. A scoreboard that displays the points of the users who receive the highest score is one of these features. If a user continues to get answers correct and goes on a "streak," their multiplier increases, encouraging them to perform better and position themselves higher on the score board. When making quizzes, Kahoot provides a wide range of customisation choices to maintain a high level of interest. (10/10) Gamification is important in code combat because it teaches the user how to write code by putting them through "levels," each of which has animations and consequences that change based on the inputs from the user. This keeps the user interested by simulating an RPG, which may be a very engaging setting and inspire young users to keep studying. (8/10) Typing.com is designed to help users improve their typing skills. The user can choose between 3 diffrent difficulties, those being "Beginner", "Intermediate" & "Advanced" the user can practice typing by playing games and typing out sentances (6/10)
Progress tracking: How well does the app track the user's progress? Does it provide feedback and recommendations to help users improve their performance? Does it use data to personalize the learning experience? Regularly updating the user's progress as the quiz advances, Kahoot displays to them the extent of their performance following each question, including the correct answer and whether or not they answered the question correctly. They are also awarded points based on how quickly they answered the question. (8/10) The amount of effort a user puts in to finish the several "levels" the program provides to assist educate a variety of codes is how progress in code fighting is measured. As the user advances, more difficult experiences become available to them, contingent on how many "levels" they have finished. Additionally, the user may buy equipment with the cash they earn by finishing levels. For the user, this is a very fulfilling experience that validates the effort they put into their educational journey. (9/10) The user has a "Daily Goal" to reach on Typing.com, the user's avrage speed, accuracy and the time spend typing at the top of the page. using the "Career Prep" section of the website can help users with their time management as well as communication & collaberation skills(7/10)
User experience: How user-friendly is the app? Is it easy to navigate and use? Does it have clear instructions and prompts? Does it have a well-designed interface? The initial Kahoot page can be a little overwhelming at first, especially with its persistent push to upgrade to a membership option. This can make it challenging for users to locate the quiz they want to take, especially if it's their first time using the app. The user experience is much easier to manage after the user is involved in a quiz since the color selections are clear and straightforward to comprehend when selecting your response. (5/10) With big, easy-to-read language to help the user better comprehend the next phase in their learning process, Code Combat is incredibly easy for the user to navigate. The user interface (UI) makes it simple for the user to maintain track of where they are by directing them to the furthest level they are in with a large animated arrow even when they click on a stage of levels. (9/10) Typing.com's website is very user friendly, all the features that can be used by the user are labled in easy to understand tabs that are clickable to begin working. (7/10)
